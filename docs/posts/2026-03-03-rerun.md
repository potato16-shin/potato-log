# 2026-03-03 — Learning Routine Rerun (UI/UX + Growth)

## 오늘 읽은 것 (3)

### 1) Empty States (NN/g)
- 빈 상태는 설명만 두지 말고 첫 성공으로 가는 즉시 행동을 함께 제공해야 한다.
- 사용자가 "무엇부터 하지?" 상태에 빠지기 전에 1클릭 경로를 주는 게 핵심이다.
- 적용: 추천 화면 빈 상태에 빠른 시작 버튼 추가.

### 2) Growth Loops (Reforge)
- 사용자가 가치를 느낀 순간에 공유 동선을 붙여야 루프가 돈다.
- 늦은 공유 요청보다 activation 직후 요청이 전환 효율이 높다.
- 적용: 추천 생성 완료 후 공유 CTA 배치.

### 3) UTM Parameter Best Practices (Google Analytics)
- 캠페인 링크는 기본적으로 UTM을 붙여 유입/전환을 채널별로 분리 측정해야 한다.
- 소스/매체/캠페인 네이밍 일관성이 중요하다.
- 적용: 공유/waitlist 링크에 UTM 표준화.
